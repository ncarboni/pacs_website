Title: Headline

----

Excerpt:

Creates a headline to group fields

----

Text:

The headline field helps to group larger sets of fields. It creates a headline with additional white-space and a prepended index number.

## Example

```
fields:
  pageSettings:
    label: Page Settings
    type: headline
  title:
    label: Title
    type: title
  draft:
    label: Draft
    type: toggle
  content:
    label: Content
    type: headline
  text:
    label: Text
    type: textarea
```
