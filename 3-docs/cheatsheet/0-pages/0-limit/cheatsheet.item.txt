Title:

$pages->limit($limit)

----

Excerpt:

Returns the collection with a limited amount of pages.

----

Return:

type: $pages
text: 

----

Text:

## Example

```php
<h2>Latest subpages</h2>
<ul>
  <?php foreach($page->children()->limit(3) as $subpage): ?>
  <li>
    <a href="<?php echo $subpage->url() ?>">
      <?php echo html($subpage->title()) ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
```
