Title: Site Visit Register

----

Description: Documentation regarding the pre-acquisition visit of the site

----

(toc:4)

(l3: Researcher Name)

Type: *Link - ((link: docs/general-register/actor-register text: Actors Register))*

(l3: Start Date)
*Indicates the initial date of the site visit*
Type: *Date*
Required: *YES*
Repeatable: *NO*

(l3: End Date)
*Indicates the end date of the site visit if multi-day.*
Type: *Date*
Required: *NO*
Repeatable: *NO*

(l3: Site Visit Documents	)
*A list of documents any documents relevant to the site visit.*
Type: *Link - ((link: docs/general-register/digital-asset text: Digital Assets Register))*
Required: *NO*
Repeatable: *YES*

(l3: Site Visit Documents Generated)
*A list of documents generated by the site visit. This field is especially envisioned to allow linking of site visit photos.*
Type: *Link - ((link: docs/general-register/digital-asset text: Digital Assets Register))*
Required: *NO*
Repeatable: *YES*
